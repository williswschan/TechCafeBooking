üìã Usage Guide

For Users

  1. Access the Portal: Navigate to the main booking page
  2. Select Date: Choose from available business days
  3. Choose Time Slot: Click on an available time slot
  4. Enter Name: Type your name (with auto-complete suggestions)
  5. Confirm Booking: Click "Book Slot" to confirm
  6. Manage Booking: Cancel your booking if needed

For Administrators

  1. Access Admin Panel: Click "Admin" button and enter password
  2. Manage Bookings: Click on booked slots to manage them
  3. Upload Files: Update display names and bad words lists
  4. Export Data: Download booking records and system files
  5. System Maintenance: Monitor and maintain the booking system
‚ö†Ô∏è Important: Admin password is required for administrative functions. Default password can be configured via environment variable.

üîå API Endpoints

Main Portal Endpoints

GET / - Main booking portal
GET /get_bookings - Get all bookings
GET /get_display_names - Get display names for type-ahead
GET /get_server_time - Get current server time

Booking Endpoints

POST /book - Create new booking
POST /cancel - Cancel existing booking
POST /admin_complete - Mark booking as completed (admin)

Admin Endpoints

GET /admin - Admin panel interface
POST /admin/verify - Verify admin password
GET /admin/download_csv - List CSV files
GET /admin/download_csv/<filename> - Download CSV file
GET /admin/download_bad_words - Download bad words file
POST /admin/upload_display_names - Upload display names
POST /admin/upload_bad_words - Upload bad words list
DELETE /admin/delete_csv/<filename> - Delete CSV file
